# 佛经 App 发布收敛清单

## 发布目标

尽快发布一个可用、稳定、安静的 1.0 版本。首版不追求经文数量和功能完整度，优先保证核心流程可信、可完成、可重复测试。

## 1.0 必须保留

- 今日功课：心经、大悲咒、阿弥陀佛计数。
- 经藏：可浏览正式经文，未接入全文的经文只能查看详情，不能进入占位阅读。
- 阅读器：字号、简繁、拼音、注释、夜间模式、书签、回到开头、单句循环、完成。
- 回向：完成后保存当日回向，避免当天重复保存。
- 我的：统计、最近回向记录、重置今日功课。

## 1.0 必须修完

- App Icon：补齐 App Store 所需图标资产。
- App Store 元数据：名称、副标题、描述、关键词、隐私说明、截图。
- 真实设备回归：普通字号和大字号各跑一遍完整流程。
- Release 构建：使用 Release 配置归档并上传 TestFlight。
- 文本说明：明确当前正式文本和待接入全文状态，避免用户误以为占位经文可读。

## 可延期到 1.1

- 更多经文全文接入。
- 书签列表和书签详情。
- 回向记录详情页。
- 日课自定义。
- 日课进度和普通阅读进度完全分离。
- 音频真实文件和时间轴。
- 多版本对照、抄经、笔记、高亮。

## 每次发布前固定验证

```bash
xcodebuild -project FoJing.xcodeproj -scheme FoJingApp -sdk iphonesimulator -destination 'generic/platform=iOS Simulator' build
```

真实设备手测路径：

1. 今日页进入心经，完成。
2. 继续大悲咒，完成。
3. 阿弥陀佛计数到目标。
4. 进入回向并保存。
5. 我的页确认记录存在。
6. 重置今日功课，确认今日页恢复。
7. 经藏搜索心经和大悲咒。
8. 大字号模式检查今日页、阅读页播放栏、回向页、我的页。
